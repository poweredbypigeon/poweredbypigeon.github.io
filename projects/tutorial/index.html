TUTORIAL

Building an online community of learners.

PROJECT DETAILS

UPDATE HISTORY

FEATURES

TUTORial is a tutoring website that connects tutors and students willing to learn, allowing them to communicate anytime, anywhere. There are 4 types of users: moderators, teachers, tutors and students. TUTORial is in the style of an online forum. Once you sign up, you can ask a question, which will be answered first and foremost by a team of student tutors. Once the user is satisfied, the question will be made public. Other students can make suggestions to improve the answer (and can add alternate solutions of their own).

PROGRAMMING DETAILS

TUTORial is coded using HTML, CSS, JavaScript, and it also uses PHP and MySQL as a database to store posts and answers.
