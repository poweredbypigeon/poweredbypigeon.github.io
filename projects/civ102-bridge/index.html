CIV102 Bridge Analysis

MATLAB code that can analyze the performance of a bridge in multiple stress modes.

PROJECT DETAILS

UPDATE HISTORY

FEATURES

CIV102's final project is to design a bridge that can withstand the weight of a train moving across it.


Desinging of the bridge requires knowledge of the material's properties, as well as calculation of the stress at every point the train traverses across it.

This MATLAB code accounts for the shear stress / flexural stress / plate buckling stress at every point
